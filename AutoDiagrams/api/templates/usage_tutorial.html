How to use AutoDiagrams

Welcome to the AutoDiagrams tool. This tool allows you to upload files (yaml, yml, json, and xml) to create an Entity Relationship diagram automatically.

Steps to use AutoDiagrams:

  1. Click on the "Upload File" button to upload a file.
  2. Choose a file from your local machine.
  3. Click on the "Upload" button to upload the file.
  4. Wait for the diagram to be generated.
  5. View the generated diagram.
  6. Download the diagram if needed.
